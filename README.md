# Разворачивание VDS
Установим:
1. Java - 17 
2. Jenkins
3. Selenoid
4. Selenoid UI


## Что надо сделать

Выьрать любой VDS хостинг и развернуть сервак

1. Создать сервер на Ubuntu 20.04.6 LTS( мин 4Gb ОЗУ)
2. Зайти под root
3. `git clone https://github.com/MaksimKazakov/JSSuiVDS.git`
4. `cd JSSuiVDS`
5. `chmod +x build1.sh`
6. `./build1.sh`
